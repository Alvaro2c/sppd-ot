Nuestra Misión

Proporcionar acceso transparente, accesible y completo a los datos de contratación pública española, empoderando a empresas, investigadores y ciudadanos para comprender y participar en las actividades de contratación gubernamental.

Transparencia

Hacer que los datos de contratación pública estén abiertamente disponibles y fácilmente accesibles para todos.

Análisis

Proporcionar herramientas simples para analizar tendencias, patrones e información de los datos de contratación.

Inclusión

Asegurar acceso igualitario a oportunidades de contratación para empresas de todos los tamaños.

Sobre Nuestros Datos

Fuente de Datos

Nuestros datos de contratación provienen de la Plataforma de Contratación del Estado (PCE), base de datos oficial del gobierno español.

Cobertura de Datos

Proporcionamos datos actualizados de licitaciones públicas de todas las comunidades autónomas de España. Nos centramos en licitaciones activas que aún aceptan ofertas y aquellas actualizadas durante los últimos 3 meses.

Frecuencia de Actualización

Los datos se actualizan diariamente para asegurar que la información más actual esté disponible.

Calidad de Datos

Todos los datos pasan por procesos de verificación de calidad y validación para asegurar precisión, completitud y consistencia.
