Preguntas Frecuentes

Encuentra respuestas a preguntas comunes sobre nuestra plataforma de Datos de Contratación Pública Española.

¿Con qué frecuencia se actualizan los datos?

Nuestros datos de contratación se actualizan diariamente para asegurar que tengas acceso a la información más actual disponible.

¿Puedo descargar los datos para mi propio análisis?

¡Sí! Puedes exportar datos en formatos CSV y JSON desde nuestra página de datos. Por el momento no tenemos acceso API para desarrolladores.

¿Los datos son gratuitos para usar?

Sí, todos los datos están disponibles gratuitamente bajo licencias de datos abiertos. Creemos en la transparencia y accesibilidad.

¿Puedo solicitar datos específicos que no están disponibles actualmente?

Siempre estamos buscando expandir nuestra cobertura de datos. Por favor contáctanos con tus requisitos específicos.

¿Qué regiones están cubiertas en los datos?

Cubrimos datos de contratación pública de las 17 comunidades autónomas y las 2 ciudades autónomas españolas, incluyendo gobierno central, gobiernos regionales y autoridades locales.

¿Hasta dónde se remontan los datos históricos?

Actualmente solo cubrimos datos de licitaciones abiertas y/o actualizadas en los dos últimos meses y el mes en curso, para centrarnos en datos recientemente actualizados. Próximamente incluiremos datos históricos para mayor análisis.

¿Hay alguna restricción de uso?

No, los datos se proporcionan bajo licencias de datos abiertos.

¿Cómo puedo obtener soporte técnico o reportar errores?

Para soporte técnico o preguntas sobre el uso de nuestra plataforma, por favor envíanos un email a alvarocarranzacarrion@gmail.com.
